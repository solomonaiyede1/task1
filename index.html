Intro

CityHome Responsive HTML Real Estate Template. Easy-to-customize responsive template. The CityHome Real Estate template allows you to modify and build unique real estate websites with ease. CityHome Real Estate Theme is the right Bootstrap template that allows you to build sites for real estate developers, real estate sellers, as well as real estate brokers.
The CityHome Real Estate Template allows developers to develop real estate sites quickly and easily. It is the perfect Real Estate template for sellers, buyers, and brokers.
The CityHome Real Estate Template is 100% responsive, so you can target customers of all categories and not restrict the business owners to customers using a specific kind of device. End users can access the site from their preferred device and have the best UX as it will adapt perfectly to the screen size of the devices. A website built using the CityHome Real Estate Template can be accessed conveniently from devices like Desktops, Laptops, iPhones, Android phones, Tablets, etc.
The CityHome Real Estate Template was built while keeping developer’s ease in mind, so that they can build beautiful websites quickly and easily. It is built with SEO Optimized Code, to ensure the site is SEO friendly. If a real estate website is built using the CityHome Real Estate Template, the site owners will have the SEO advantage and be able to beat the market competition to a great extent.
This CityHome Real Estate Template is meant for offering a high-grade user experience to all its users. It offers a faster page-loading time along with superior quality user experience (UX).

Include Script

First, include js script into your HTML footer script:

<script>
        var form = document.getElementById("my-form-contact");

        async function handleSubmit(event) {
            event.preventDefault();
            var status = document.getElementById("my-form-status-contact");
            var data = new FormData(event.target);
            fetch(event.target.action, {
                method: form.method,
                body: data,
                headers: {
                    'Accept': 'application/json'
                }
            }).then(response => {
                status.innerHTML = "Thanks for your submission!";
                form.reset()
            }).catch(error => {
                status.innerHTML = "Oops! There was a problem submitting your form"
            });
        }
        form.addEventListener("submit", handleSubmit)
    </script>
